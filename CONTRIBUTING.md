# Contributing

Thanks for contributing to Ouroboros Snake Strategy.

This repo is the flagship public ensemble in the Glitch trading family. The best contributions keep the Oracle-plus-six-snake story clear, the docs navigable, and the public implementation safely sanitized.

## Good Contribution Targets

- architecture documentation
- operating-model clarity
- repo navigation and diagrams
- sanitized config examples
- portability improvements across MT5 and cTrader tracks

## Please Avoid

- committing credentials, logs, state, or model artifacts
- mixing private operating details into public docs
- changing bot identities or family structure without documenting the rationale

## Workflow

1. Branch from `main`.
2. Keep changes scoped and easy to review.
3. Update docs alongside any public structure or naming changes.
4. Open a PR with a short summary, validation, and ecosystem impact notes.

## Public Repo Hygiene

- configs in Git are templates only
- secrets and runtime state stay outside version control
- preserve license, notice, and authorship files
